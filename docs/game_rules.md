# Taiwan Brawl 遊戲規則

最後更新：2026-04-22

本文以目前後端與前端實作為準，主要來源是 Royale 對戰、卡牌、養成與場地事件邏輯。

## 1. 對戰基本規則

- 每場對戰時間為 `210` 秒。
- 牌組固定 `8` 張卡，且同一副牌組不能放重複卡。
- 每次最多同時打出 `3` 張卡。
- 每張卡在單局內預設最多使用 `8` 次。
- 卡牌次數用完後，不會再回到手牌循環。
- 左方部署區域為 `0-420`，右方部署區域為 `580-1000`。
- 雙人對戰需要雙方都 ready，且 websocket 已進入遊戲，才會開始。
- 玩家斷線有 `15` 秒緩衝，超過後判對方勝利。
- 勝利條件是敵方主堡血量歸零。
- 時間到時，由剩餘主堡血量較高者勝。
- 時間到且雙方血量相同時，判定平手。

## 2. 角色與主堡血量

目前主堡實際上已接近角色本體。戰鬥中的主堡血量採雙軌制：

- 身體血量：`physicalHealth`
- 精神血量：`spiritHealth`

死亡判定：

- 身體血量歸零時，角色死亡。
- 精神血量歸零時，角色死亡。
- 任一條血量歸零就會輸掉該局，不需要兩條都歸零。

對外快照中仍保留 `towerHp` 與 `maxTowerHp`，主要是為了相容既有 UI 與歷史資料。`towerHp` 目前仍是身體血量與精神血量的總和，但勝負判定以雙軌血量為準。

## 3. 戰鬥資源

戰鬥中有三種主要資源：

- 體力：`physicalEnergy`
- 精神：`spiritEnergy`
- 金錢：`money`

卡牌消耗規則：

- 單位卡預設消耗體力。
- 法術/技能卡預設消耗精神。
- 裝備卡預設消耗金錢。
- 事件卡和打工卡可以指定自己的消耗類型。
- 物理傷害只扣身體血量。
- 精神傷害只扣精神血量。
- 傷害不會從一條血量溢出到另一條血量。

## 4. 英雄類型

目前實作中的英雄如下：

- 富二代：初始金錢高，但精神疾病事件更容易觸發。
- 普通人：精神疾病較難觸發，但一旦觸發會從第二階段開始。
- 中低收入戶：初始金錢較少，但單位傷害提升。
- Part time 打工仔：工作事件高報酬機率較高，但負面結果也較不穩定。

英雄會影響：

- 初始身體血量、精神血量、體力、精神、金錢。
- 單位傷害倍率。
- 工作事件收入倍率。
- 工作事件正面或負面權重。
- 精神疾病觸發與傷害。

## 5. 卡牌類型

### 5.1 單位卡

- 部署到場上後會自動移動、尋敵、攻擊。
- 進入攻擊距離後停止移動並攻擊。
- 單位會依目標方向切換面向。
- 攻擊動畫以實際出手事件觸發，不是只要在攻擊範圍內就連續播放。

### 5.2 法術/技能卡

- 對指定範圍造成傷害。
- 可傷害範圍內敵方單位。
- 若施放範圍覆蓋敵方主堡，也可傷害主堡。

### 5.3 裝備卡

- 一般裝備需要跟至少一張單位卡一起打出。
- 裝備效果會套用到同次部署的單位。
- 主角裝備卡可以直接作用在自己角色身上。

### 5.4 事件卡

- 事件卡必須單獨打出。
- 不能和單位、裝備、法術或打工卡一起組合施放。

### 5.5 打工卡

- 打工卡必須單獨打出。
- 打出後會觸發隨機工作事件。
- 結果可能增加金錢，也可能扣身體、精神或能量。

## 6. 部署與手牌循環

- 每位玩家開局從牌組抽出前 `4` 張作為手牌。
- 使用卡牌後，若該卡還有剩餘使用次數，會回到牌庫隊尾。
- 若該卡已達單局使用上限，會從循環中移除。
- 每次使用後會從隊列補牌，維持手牌數量。

## 7. 單位戰鬥規則

- 單位會朝敵方方向推進。
- 單位會優先攻擊前方可攻擊範圍內的敵方單位。
- 若沒有合適單位目標，且主堡在攻擊距離內，會攻擊主堡。
- `targetRule = tower` 的單位會優先直衝主堡。
- 攻擊距離會計入單位 body radius 與主堡 body radius。
- 單位死亡後會從場上移除。

## 8. 主角攻擊規則

主角目前有防守型自動攻擊能力：

- 主角只會攻擊靠近自己主堡/角色位置的敵方單位。
- 主角不會直接攻擊敵方主角。
- 主角攻擊有自己的傷害、射程、攻擊間隔與傷害類型。
- 主角攻擊會走自己的 cooldown，不消耗手牌或戰鬥資源。
- 主角攻擊會產生 `heroAttackEvent`，之後可用來對齊主角攻擊動畫。

目前主角攻擊數值：

- 富二代：精神傷害 `48`，射程 `260`，攻擊間隔 `1.35` 秒。
- 普通人：物理傷害 `64`，射程 `230`，攻擊間隔 `1.25` 秒。
- 中低收入戶：物理傷害 `76`，射程 `205`，攻擊間隔 `1.15` 秒。
- Part time 打工仔：物理傷害 `58`，射程 `220`，攻擊間隔 `0.95` 秒。

## 9. 狀態效果

目前單位可承受的狀態效果：

- 瘀青：持續造成每秒 `20` 傷害，持續 `3` 秒。
- 流血：持續造成每秒 `50` 傷害，持續 `4` 秒。
- 精神疾病：攻擊間隔增加，持續 `5` 秒。
- 暈眩：無法移動與攻擊，持續 `0.9` 秒。
- 緩速：移動速度變為 `70%`，持續 `2` 秒。

## 10. 角色養成系統

每個牌組都有自己的培養角色。

目前可選的牌組角色：

- 普通小孩
- 街頭小聰明
- 讀書苦行
- 任一張單位卡

規則：

- 初始角色只能在該牌組尚未開始成長前更換。
- 選擇單位卡作為牌組角色時，角色 id 會直接使用該卡牌 id。
- 法術、裝備、事件、建築與打工卡不能作為牌組角色。
- 勝利會增加年齡。
- 平手會增加 `1` 歲。
- 失敗不增加年齡。
- 輸掉會扣健康度。
- 健康度每天自動恢復 `6`。
- 健康度歸零時會重生。
- 重生後年齡歸 `0`，健康度回 `100`，重生次數加 `1`。

勝利增加年齡：

- 小勝：`+1` 歲。
- 中勝：`+2` 歲。
- 大勝：`+3` 歲。

失敗扣健康度：

- 普通敗北基礎扣 `14`。
- 斷線敗北基礎扣 `25`。
- 會再依主堡受損比例追加懲罰。

每場結束後會記錄：

- 勝敗結果。
- 年齡變化。
- 健康度變化。
- 是否重生。
- 該局各卡牌使用次數。
- 該牌組累積天賦歷程。

## 11. 年齡解鎖

目前實作中的卡牌解鎖年齡：

- `0` 歲：道具/裝備卡。
- `3` 歲：事件卡。
- `5` 歲：技能/法術卡。
- `8` 歲：一般單位卡。
- `14` 歲：青少年單位，例如 `8+9`。
- `16` 歲：打工卡。

目前狀態：

- API 與 UI 已會回傳和顯示解鎖年齡。
- 牌組儲存尚未硬性阻擋未解鎖卡。
- 暫時不硬擋的原因是避免 0 歲角色只能用裝備卡，導致既有牌組無法開局。

## 12. 特殊卡牌

### 12.1 檳榔

- 類型：裝備卡。
- 費用：`$5`。
- 套用到單位時，提升移動速度與攻速。
- 套用到單位時，會讓單位持續扣血。
- 套用到主角時，增加身體血量上限、目前身體血量與體力。
- 套用到主角時，降低身體回血，並增加癌症風險紀錄。

### 12.2 瓜皮安全帽

- 類型：裝備卡。
- 費用：`$70`。
- 可作用在主角。
- 效果是恢復或增加身體血量。

### 12.3 明星花露水

- 類型：裝備卡。
- 費用：`$12`。
- 可作用在主角。
- 效果是恢復精神血量。

### 12.4 路邊老人

- 類型：單位卡。
- 目標規則：主堡。
- 打到主堡時，會偷對方低價道具的一次剩餘使用次數。
- 低價道具目前包含瓜皮安全帽、雨傘、檳榔、明星花露水。

### 12.5 8+9

- 類型：青少年單位。
- 解鎖年齡：`14` 歲。
- 以群體單位方式部署。

### 12.6 媽媽砸摳

- 類型：事件卡。
- 效果：立刻獲得 `$10`。

### 12.7 豬大哥還沒死

- 類型：事件卡。
- 效果：反轉目前所有卡牌的使用次數與剩餘次數。
- 例如單局上限 `8` 次，某卡已使用 `5` 次且剩 `3` 次，使用後會變成已使用 `3` 次且剩 `5` 次。

### 12.8 天氣

- 類型：事件卡。
- 效果：立刻推進下一個場地事件。

## 13. 場地事件

- 場地事件每 `30` 秒觸發一次。
- 事件會保留在戰鬥事件列表中，最多顯示最近 `6` 筆。
- 部分事件是一次性效果。
- 部分事件會留下持續場地效果。

目前事件類型包含：

- 交通事故：山道猴子、馬路三寶、酒癮慣犯、惡意肇逃、大卡車、雷殘事故。
- 治安：詐騙集團猖獗。
- 政治：世界大戰爆發。
- 家庭：蟑螂藥下毒、全區停電、全區停水、恐龍家長、亞洲家長。
- 公司：好老闆犒賞、慣老闆強制加班。
- 回復：強制戒毒療程、緊急住院。
- 食品：食品中毒。
- 外送：外送尖峰潮。

恐龍家長事件：

- 會隨機保護一方。
- 被保護的一方可抵擋下一次負面場地事件。

## 14. 工作事件

目前工作卡包含：

- 超商班。
- 外送單。
- 粗工現場。

工作事件會依卡牌類型抽取結果：

- 可能增加金錢。
- 可能扣身體血量。
- 可能扣精神血量。
- 可能扣體力或精神能量。
- 可能觸發精神疾病階段。

英雄會影響工作事件：

- 富二代較容易受到精神事件影響。
- 普通人精神事件較難觸發，但觸發後階段較高。
- Part time 打工仔較容易出現高報酬，也更容易出現不穩定結果。

## 15. 圖像與動畫資源

目前卡牌/角色圖像支援：

- 卡牌主圖。
- 角色正面圖。
- 角色背面圖。
- 角色朝左圖。
- 角色朝右圖。
- 多張動態角色素材。
- 每張動態素材可設定 animation、direction、frameIndex、durationMs、loop。

部署到戰場上的單位會帶出：

- `characterAssets`
- `characterImageUrls`
- `facingDirection`
- `animationState`
- `animationEvent`

目前攻擊動畫以事件型 `animationEvent` 觸發，適合對齊每次實際出手。
